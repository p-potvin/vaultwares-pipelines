# Contributing to VaultWares Pipelines

Thank you for helping strengthen the core AI engine of the VaultWares ecosystem.

## Mandatory Steps

1. Read the [Brand Guidelines](https://raw.githubusercontent.com/p-potvin/vaultwares-docs/main/agents/branding.mdx) and [Agent Knowledge Dissemination System](https://raw.githubusercontent.com/p-potvin/vaultwares-docs/main/agents/knowledge-dissemination.mdx).
2. Follow the exact structure and naming from [vaultwares-template](https://github.com/p-potvin/vaultwares-template).
3. All new pipelines must be defined in YAML with full agent hooks.
4. Test locally and with the coordinated agent system before submitting a PR.
5. Maintain local-first privacy in every pipeline step.

See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) for community standards.
