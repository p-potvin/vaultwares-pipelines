# VaultWares — Agent Quick Brief

You are operating inside a **VaultWares** repository. Before any UI, branding, theme, design-token, or user-facing string change, **read the canonical sources** in this order:

1. `vault-themes/AGENTS.md` — top rules, security posture, theme catalogue.
2. `vault-themes/brand/ui-kit.md` — **product-UI vocabulary** (this is the most important one for any UI task).

If the `vault-themes` submodule isn't checked out in this repo, fetch it before proceeding. If it cannot be fetched, ask the user before making any visual decision.

## Non-negotiables

- **VaultWares products are instruments, not websites.** Default to: app shell · sidebar · list with hairline dividers · right-rail drawers · status bar. *Not*: card grids, hero panels, marketing CTAs, bubble pills.
- **Tokens, not hex.** All colors, spacing, fonts, radii, motion, and glass values come from `vault.*` tokens. Never hardcode.
- **Bilingual parity.** Every visible string has both `en` and `fr` (Quebec French). FR runs +15–20% longer — layouts must tolerate it.
- **No emoji in product/UI/marketing copy.** No unicode glyphs as icons. Use Lucide.
- **No glass on full pages.** Glass is reserved for command palettes and rare focused overlays.
- **No fear copy.** Forbidden: "military-grade", "hacker-proof", "bulletproof", "Proceed", "CRITICAL SECURITY WARNING", "just", "simply", "easy" (unless factually proven). Replace with calm, technically honest language.
- **Privacy first.** Never design a flow where the server can read, persist, derive, or reconstruct private keys or shared decryption keys used between two clients. ML-KEM is the post-quantum KEM target.
- **Accessibility.** Body contrast ≥ 4.5:1 (WCAG AA). Large/UI ≥ 3.0:1. Focus visible in both modes. State never relies on color alone.

Gold = brand. Cyan = interaction. Green = secured. Burgundy = destructive. **They never swap roles.**

## Voice anchors

- Tagline: *"Privacy first. Security in service."* / *"La confidentialité d'abord. La sécurité au service."*
- Replace *"Encrypted with military-grade AES-256"* → *"Vault secured."*
- Replace *"Critical security warning"* → *"We noticed something. Here's what to do."*
- Replace *"Proceed"* → *"Continue."*

## Anti-patterns to reject on sight

Pure black backgrounds · neon green · Matrix rain · padlock illustrations · multi-stop gradients · backdrop-filter on full pages · emoji as icons · cards with gradient backgrounds or left-accent stripes · "AI-coded" rainbow chrome · infinite decorative motion loops · hardcoded hex outside `tokens.ts` · EN strings without FR counterparts.

## When in doubt

Ask the user. Cite the source file you read. Do not invent values, do not guess colors, do not synthesize French translations — read `brand.i18n.ts` or ask.
