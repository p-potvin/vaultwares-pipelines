## Voice

Calm, precise, human, principled, competent. Bilingual EN / Quebec FR parity (FR runs +15‚Äì20%). Tagline: *"Privacy first. Security in service."* Sentence case for buttons.

Avoid: "just / simply / easy", "military-grade", "hacker-proof", "Proceed", emoji, neon green on black, terminal aesthetics, padlock-with-lightning illustrations. Replace "Proceed ‚Üí Continue", "Critical security warning ‚Üí We noticed something. Here's what to do.", "Encrypted with military-grade AES-256 ‚Üí Vault secured."

## Icons

Lucide is the default. Mintlify docs reference FontAwesome by name (`shield-halved`, `microchip`, `usb-drive`, `wand-magic-sparkles`) ‚Äî substitute Lucide for visual consistency. **No emoji**, **no unicode-glyph icons**.
